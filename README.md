# CipFilm Scraper

  A web scraper app for klikfilm.web.id built with React + Vite and Express.

  ## Features
  - Browse movies scraped from klikfilm.web.id  
  - Search by title
  - Filter by genre
  - Movie detail pages

  ## Tech Stack
  - Frontend: React + Vite + TailwindCSS
  - Backend: Express.js with cheerio scraper
  - Deploy: Vercel
